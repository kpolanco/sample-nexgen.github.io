# NexGen Cloudware Samples
Bienvenido al repositorio de **NexGen Cloudware Samples** (`https://sample.nexgencloudware.com/`), un espacio público destinado a la exhibición de **prototipos de páginas web, landing pages y sitios informativos** desarrollados como referencia para clientes actuales, prospectos y proyectos comerciales.

---
## Descripción
Este repositorio almacena **contenido estático de prototipos web**, diseñados para demostrar capacidades de diseño, experiencia de usuario (UX/UI) y estructura de soluciones digitales que NexGen Cloudware puede ofrecer.

Los prototipos aquí publicados tienen como objetivo:

- Presentar ideas de **landing pages modernas**
- Mostrar diferentes estilos visuales y enfoques de marca
- Validar conceptos con clientes antes del desarrollo final
- Servir como base para futuros proyectos personalizados
- Acelerar procesos comerciales y de preventa

Todos los sitios se despliegan automáticamente mediante **GitHub Pages** en el dominio:

https://sample.nexgencloudware.com/

---
## Características
- **Repositorio público de demostración**
  Accesible para clientes, prospectos y equipo interno.

- **Contenido 100% estático**
  HTML, CSS, JavaScript y assets optimizados para hosting estático.

- **Diseños reutilizables**
  Cada prototipo puede evolucionar hacia un proyecto productivo.

- **Enfoque UX/UI**
  Experiencias modernas, responsivas y orientadas a conversión.

- **Despliegue automático**
  Integrado con GitHub Pages para publicación inmediata.

---
## Tipos de Prototipos
En este repositorio podrás encontrar ejemplos de:

- Landing pages corporativas

## Flujo de Publicación
1. Clonar el repositorio:
   git clone https://github.com/tu-usuario/nexgen-samples.git
2. Agregar o modificar prototipos
3. Subir cambios:
   git add .
   git commit -m "Nuevo prototipo"
   git push origin main
4. GitHub Pages desplegará automáticamente 🚀
